# VinMeld Tap

## How do I install these formulae?

`brew install vinmeld/tap/<formula>`

Or `brew tap vinmeld/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "vinmeld/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
